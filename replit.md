# 바이낸스 테스트넷 선물 트레이딩 봇

## 개요
BTCUSDT 선물 자동 트레이딩 봇으로, 바이낸스 테스트넷에서 가상 자금으로 거래 연습을 합니다.

## 주요 기능
- **EMA 크로스오버 전략**: EMA20/EMA60 크로스오버 기반 매매
- **RSI 필터**: RSI 과매수/과매도 필터링
- **트레일링 스탑**: 1.5% 트레일링 스탑 자동 설정
- **하드 스탑로스**: -5% 손실 시 자동 청산
- **격리 마진**: 안전한 격리 마진 모드
- **레버리지 1배**: 안전한 1배 레버리지

## 거래 조건
### 롱 진입
- EMA20 > EMA60
- 가격 > EMA20
- RSI < 68

### 숏 진입
- EMA20 < EMA60
- 가격 < EMA20
- RSI > 32

## 설정값
- **포지션 비율**: 10% (잔고의 10%로 진입)
- **타임프레임**: 15분봉
- **트레일링 스탑**: 1.5%
- **하드 스탑로스**: -5%

## 환경변수 (Secrets)
- `API_KEY`: 바이낸스 테스트넷 API Key
- `API_SECRET`: 바이낸스 테스트넷 Secret Key

## 기술 스택
- Python 3.11
- binance-futures-connector (공식 라이브러리)
- pandas (기술적 분석)
- Flask (상태 모니터링 웹서버)
